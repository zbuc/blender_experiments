# GitHub Pages - E2E Test Results

This directory contains the GitHub Pages site for displaying End-to-End (E2E) test results.

## Files

- `e2e-viewer.html` - Main HTML page for viewing test results
- `e2e-results/` - Test results data (generated by CI)
  - `index.json` - Index of all test cases
  - `{test_name}.json` - Individual test case data
  - `images/{test_name}/` - Test images (reference, rendered, debug)

## Local Development

To view locally:

1. Run E2E tests to generate results:
   ```bash
   cd blender_blocking
   blender --background --python test_e2e_validation.py
   ```

2. Serve the docs directory:
   ```bash
   cd docs
   python3 -m http.server 8000
   ```

3. Open http://localhost:8000/e2e-viewer.html

## GitHub Pages Deployment

The site is automatically deployed via GitHub Actions when tests run on the `main` branch.

View the live site at: `https://zbuc.github.io/blender_experiments/`
